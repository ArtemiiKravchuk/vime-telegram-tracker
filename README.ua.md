# Телеграм трекер
![Static Badge](https://img.shields.io/badge/версія-2-blue)
![Static Badge](https://img.shields.io/badge/python-3.10-orange)

Спострігайте за змінами в публічних даних користувачів Телеграм, як-от профілі, історії, або статус онлайн.

## Встанолвення

Проклонуйте репо з GitHub та встановіть бібліотеки:

``` bash
git clone https://github.com/ArtemiiKravchuk/vime-telegram-tracker.git
pip install -r requirements.txt
```

## Конфігурація

Змініть `config.json` до ваших потреб.

Потрібно заповнити поля `telegram => api_id` та `telegram => api_hash` значеннями з from https://my.telegram.org

В полі `targets => users`, розмістіть список користувачів. Ви можете використовувати цілочисленний айді (тоді користувач повинен бути в контактах) або строку-юзернейм (користувач може і не бути в контактах, але програма працюватиме повільніше)

Програма зберігає всі дані в бажаному форматі. Ви можете це змінити в полі `output` конфігурації: там повинен бути список всіх локації для виводу інформації.

Для прикладу, програма буде зберігати дані до sqlite бази даних та csv файлів, але ви можете прибрати непотрібну локацію.

## Запустити програму

Для запуску програми, запустіть `main.py`.

Програма під час роботи створить додаткові папки:
- `logs` - ви можете її видалити в будь-який момент, вона зберігає логи програми, які можуть потім допомогти вирішити проблеми.
- `output` - тут ви зможете знайти всі згенеровані програмою дані.
- `sessions` - зберігає файли `.session`: якщо ви видалите цю папку, вам потрібно буде входити в Telegram акаунт заново; після видалення цієї папки програма також може працювати повільніше.
- `temp` - не рекомендовано видаляти/змінювати цю папку, вона запобігає даним-дублікатам. Але ви можете зробити чистий запуск заново, видаливши цю папаку разом з папкою `output`.

## Розуміння виводу

Кожен модуль зберігає свої дані (профілі, історії, тощо). Ці дані зберігаються в своїх файлах (`profile.csv`, `story.csv`) або як SQLite таблиці.

Ось деякі речі, яки вам можуть знадобитися:
- Час збережений в форматі UNIX timestamp.
- Медіа збережено в папці `output`. Ви знайдете путь до медіа файлів в ваших таблицях
